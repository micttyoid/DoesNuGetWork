# DoesNuGetWork: Simple Connectivity Test

## Upload

1. Pack
    ```
    $ dotnet pack
    $ cp ./bin/Release/LukeYoo.DoesNuGetWork.*.nupkg ~/Desktop/
    ```

2. Upload at the NuGet Website
- Go to: https://www.nuget.org/packages/manage/upload
- Drop(Upload) from _Desktop_
- Submit
